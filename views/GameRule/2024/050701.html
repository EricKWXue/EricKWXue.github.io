          苏州某地区娱乐麻将玩法

          Eric Xue 2024-05-07 game rules mahjong

          # 一. 牌数

          总共152张牌，其中有108张正牌、16张风牌、28张花牌。

          • 筒、万、条 各9*4张，合计108张
          • 东、南、西、北 各4张，合计16张
          • 红中、发财、白皮 各4张，合计12张（花牌）
          • 梅、兰、竹、菊 各1张，合计4张（花牌）
          • 春、夏、秋、冬 各1张，合计4张（花牌）
          • 老鼠、财神、猫、聚宝盆各1张，合计4张（花牌）
          • 百搭 4张（花牌）

          # 二. 牌型

          • 顺子：3张同花色序数相连的牌（如：123）
          • 刻子：3张相同的牌（如：111）
          • 对子：2张相同的牌（如：11）
          • 明杠（算1个花）：分为先碰下来，再杠和一次杠两种
          • 暗杠（算2个花）：自己摸到四张相同的牌杠下来
          • 风杠（明杠：算2个花，暗杠：算3个花）：东、南、西、北风的杠

          # 三. 打法

          发牌：不带花共124张。

          打法：可碰，可杠，不可吃。

          玩法：庄家投骰子，按点数从庄家开始逆时针数，2个骰子中点数小的做切桩拿牌。庄家一开始可以拿14张牌，其它玩者只能取13张牌。发牌结束后，有几张花补几张牌从后面牌墙最后拿牌，由庄家先打出第一张牌，打牌正式开始。

          杠牌和补牌都是从最后拿牌。

          暗杠上面的一张掀开给别的玩家看到。

          记分：吃炮费用由放炮者一人承担，如果是自摸胡牌将收3家钱。

          庄家：谁胡牌谁下一把坐庄。

          诈胡：诈胡者向其他3家付相同费用（事先说好费用多少）。

          第一把随机选庄，4家投骰子，点数最大的做庄家。

          # 四. 和牌方式

          苏州麻将中各种和牌方式是累加的。和牌方式有下面几种：

          # 4.1 普通和牌

          什么是门清？ 没有碰过，也没有明杠。即，所有的牌都是自己摸到的，不存在外来的牌。

          • 门清情况下：符合ABC*x+DDD*y+FF即可。
          • 非门清情况下：3花自摸4花冲（现在也流行：2花自摸3花冲）。即，3张花及以上才能自摸，4张花以上才能和牌。

          # 4.2 特殊和牌

          • 混一色：只有筒、万、条其中的一种，以及东、南、西、北风。加5个花
          • 清一色：只有筒、万、条其中的一种。加10个花
          • 对对胡：DDD*y+FF，除了头子是一对以外，另外的都是三个头（型如：11+111+111+111+111）。加5个花
          • 七小对：成牌时，手上全部是对子，共有七对（型如：11+11+11+11+11+11+11）。加5个花（注意：七对成牌后不计门清的花）
          • 杠开：当抓到花或是杠牌后，应该在拿牌的另一侧补上一张牌，如果补上一张牌后，正好和牌，就叫做杠后开花。加5个花
          • 大吊车：其他牌已经碰下或者杠下，独听最后手中的一张牌。加5个花

          # 五. 结算

          1. 算花的部分，各地区不同，可重新定义
          2. 按花的数量，具体一个花算多少钱也可以根据富裕程度自行商量。
